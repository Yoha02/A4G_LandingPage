## Project Overview: AI-mmunity

AI-mmunity is a multi-agent system for community health built with Google ADK (Agent Development Kit). The system transforms how public health data is accessed and utilized by citizens, health practitioners, and public officials.

**Key Features:**
- Natural language access to health data dashboards
- 8 specialized AI agents coordinated by a root agent
- Real-time air quality, disease outbreak, and health guidance
- Automated PSA generation and social media outreach
- Built in just 3 days using Google Cloud technologies

**Tech Stack:** ADK, Gemini, BigQuery, Cloud Run, Veo, Firebase

**Live Demo:** [aimmunity.io](https://aimmunity.io/)
